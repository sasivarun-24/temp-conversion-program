# Temperature Conversion Program 🌡️

A simple, fast web utility to convert temperatures between Celsius and Fahrenheit.

## Features
- **Two-way Conversion:** Easily convert from Celsius to Fahrenheit (`c2f`) and vice-versa (`f2c`).
- **Instant Results:** Enter a number, select your desired conversion direction, and click submit to get immediate results.
- **Clean Interface:** A straightforward, no-nonsense UI built with HTML and CSS.

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript

## How to Run
1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/sasivarun-24/temp-conversion-program.git
   ```
2. Open `index.html` in your web browser.
3. Enter a temperature value, select your conversion direction with the radio buttons, and click **submit**.
